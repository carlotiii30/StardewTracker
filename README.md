# 🐔 Stardew Tracker

> The ultimate pocket guide for your farm. Designed to replace the 20 open wiki tabs with a beautiful, interactive interface.

## ✨ Key Features

* **🎮 Console Friendly:** No save file uploads required. Perfect for Switch, Xbox, and PlayStation players who can't export their saves.
* **📱 Offline Ready:** Works without internet as a PWA (Progressive Web App). Install it directly to your phone's home screen.
* **💾 Persistent:** Your progress (bundles, gifts, museum) is saved automatically to your browser using Local Storage.
* **🎨 Cozy UI:** Custom pixel-art interface designed to match the game's aesthetic, providing an immersive experience.

## 🛠️ Tech Stack

Built with performance and scalability in mind:

* **Core:** [React](https://react.dev/) + [Vite](https://vitejs.dev/)
* **Language:** [TypeScript](https://www.typescriptlang.org/)
* **State Management:** [Zustand](https://github.com/pmndrs/zustand) (with Persist middleware)
* **Styling:** CSS Modules
* **PWA:** vite-plugin-pwa

## 🚀 How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/tu-usuario/stardew-companion.git](https://github.com/carlotiii30/StardewTracker.git)
    cd StardewTracker
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Start the development server:**
    ```bash
    npm run dev
    ```
